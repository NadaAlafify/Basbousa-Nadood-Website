# Basbousa Nadood - Eastern Dessert Multi-Page Website 

##  Description
**Basbousa Nadoud** is a premium, custom-styled multi-page website dedicated to traditional Egyptian Basbousa. The platform serves a dual purpose: an educational hub to teach authentic dessert recipes and a commercial storefront for ordering homemade treats. 

The project features a warm, brand-specific color palette and a modern architecture spanning multiple dedicated sub-pages built entirely from scratch.

##  Live Demo
Explore the live website here: (https://nadaalafify.github.io/Basbousa-Nadood-Website/)

##  Tech Stack
- **HTML5:** Semantic architecture to manage multiple linked pages, structured content flow, and forms.
- **CSS3:** Custom responsive styling, custom typography alignment, button animations, and custom color themes designed specifically for the food industry.

##  Key Pages & Features
- **Multi-Page Navigation:** Seamless routing between Home (`الرئيسية`), Store (`المتجر`), and Recipes (`الوصفات`).
- **Hero Section:** Catchy tagline with prominent Call-to-Action buttons (`شوف المنيو` / `اتعلم السر`) and an instant order shortcut (`اطلبي الآن`).
- **Dual-Purpose Platform:** Bridging the gap between an educational recipe blog and a fast-loading commercial store.
- **Pure CSS Layout:** Built 100% manually to master multi-page link architecture and layout styling without external frameworks.
